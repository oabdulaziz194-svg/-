# -📝 تصنيف الأخبار العربية باستخدام NLP و Machine Learning
مشروع متكامل لتحليل وتصنيف النصوص العربية آلياً بدقة تصل إلى 95%

📌 نظرة عامة (Project Overview)
يهدف هذا المشروع إلى بناء نموذج تعلم آلي (Machine Learning) قادر على فهم وتصنيف محتوى الأخبار باللغة العربية. تم تدريب النموذج على مجموعة بيانات ضخمة وحقيقية لضمان قدرته على التعامل مع صياغات إخبارية متنوعة، مع التركيز على قطاعات العلوم والتكنولوجيا وريادة الأعمال.

🏗️ هيكلية العمل (Pipeline)
1. معالجة البيانات (Data Preprocessing)
نظراً لتعقيد اللغة العربية، تم تطبيق خطوات تنظيف دقيقة تشمل:

Normalization: توحيد أشكال الحروف (مثل: أ، إ، آ -> ا) و (ة -> ه).

Noise Removal: إزالة الروابط الإلكترونية، الرموز الخاصة، والتشكيل.

Stop-words Filtering: بناء قائمة كلمات توقف مخصصة لحذف الكلمات التي لا تحمل قيمة تصنيفية (مثل: من، إلى، عن).

2. استخراج الميزات (Feature Extraction)
تم استخدام تقنية TF-IDF (Term Frequency-Inverse Document Frequency) لتحويل النصوص إلى قيم عددية، مع تفعيل خاصية N-grams (1, 2) لتمكين النموذج من فهم المصطلحات المركبة مثل "الذكاء الاصطناعي".

3. النمذجة (Modeling)
تم اختيار خوارزمية Multinomial Naive Bayes لكفاءتها العالية في تصنيف النصوص (Text Classification) وقدرتها على التعامل مع المتجهات ذات الأبعاد العالية الناتجة عن النصوص العربية.

📊 النتائج والتقييم (Results & Evaluation)
مصفوفة الارتباك (Confusion Matrix)
تم تقييم النموذج باستخدام مصفوفة الارتباك للتأكد من دقة التنبؤ لكل فئة على حدة:

الدقة الإجمالية (Accuracy): ~95%

أداء الفئات: أظهر النموذج قدرة فائقة على التمييز بين الأخبار التقنية وأخبار ريادة الأعمال، مع وجود هامش خطأ بسيط جداً ناتج عن تداخل المصطلحات المشتركة بين القطاعين.
